📘 2. 專案簡介

本專題使用 Dev C++ 與 SFML 函式庫 製作一款打地鼠遊戲（Whack-A-Mole）。
玩家可選擇不同的 版面大小 及 遊戲難度，並需要在時限內擊中正確類型的地鼠以獲得最高分。

🛠️ 3. 系統需求

Dev C++：4.9.2 版本

SFML：2.4.2 版本

🕹️ 4. 遊戲特色與功能
✔ (1) 可自訂遊戲版面尺寸

3 × 3

4 × 4
➡ 玩家可在遊戲開始時自由選擇盤面大小。

✔ (2) 三種遊戲難度
難度	特點
Easy	地鼠出現時間較長、速度慢
Normal	速度中等、反應時間適中
Hard	地鼠快速變換、反應時間極短

➡ 遊戲開始時可自行選擇難度。

✔ (3) 多種類型地鼠
類型	效果
⭐ 超好地鼠	高分加成（+3 分）
👍 好地鼠	一般得分（+1 分）
👎 壞地鼠	扣分（–1 分）
💣 炸彈	觸碰將扣大量分數（–3 分）
⏱ 時鐘	增加遊戲剩餘時間（+1 秒）
✔ (4) 即時資訊顯示

遊戲畫面會即時顯示：

🟦 目前分數 (Current Score)

🟦 歷史最高分 (Best Score)

🟦 剩餘時間 (Time Left)

🟦 離開按鈕 (Exit)

✔ (5) 遊戲結束畫面

遊戲結束後會顯示：

✅ Success：成功達成指定分數

❌ Failure：未達成指標分數

🔄 Press R 重新開始

🚪 Press Q 離開遊戲
