2.專案簡介：
本專題使用 Dev C++ 與 SFML 函式庫製作一款改良版 打地鼠遊戲 (Whack-A-Mole)。
玩家可選擇不同的版面大小（3*3/4*4)及遊戲難度，並需要在時間內擊中正確的地鼠類型以獲得高分。
3.遊戲特色與功能：
(1).可自訂遊戲板尺寸
A. 3 × 3
B. 4 × 4
(2).可自行選擇遊戲難度
A. Easy
B. Normal
C. Hard
